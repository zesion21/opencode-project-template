# PPT 设计快速参考指南

这是一份精简的速查表，帮助你快速应用设计原则。

## 🎯 核心原则速记

### CRAP 原则
- **C**ontrast（对比）：大小、粗细、颜色制造差异
- **R**epetition（重复）：统一字体、颜色、样式
- **A**lignment（对齐）：所有元素精确对齐
- **P**roximity（亲密性）：相关信息靠近

### 黄金比例
- **60-30-10 配色法则**：主色60% + 副色30% + 强调色10%
- **1-3 文字层级**：最多3个层级
- **3-5 要点数量**：每页最多5个要点
- **1.2-1.4 行间距**：舒适的阅读体验

## 📐 尺寸规范

### 常用比例
- **16:9** (960x540px)：标准宽屏，最常用
- **4:3** (960x720px)：传统比例
- **A4纵向** (210x297mm)：打印用

### 字号建议
- **标题**: 36-54pt
- **副标题**: 24-32pt
- **正文**: 18-24pt
- **注释**: 14-16pt

### 间距建议
- **页边距**: 页面宽度的 5-8%
- **行间距**: 1.2-1.4 倍
- **元素间距**: 至少 20-40px

## 🖼️ 图片处理速查

### 蒙版透明度
- **轻度蒙版**：30-40%（图片可见，文字清晰）
- **中度蒙版**：50-60%（平衡效果）
- **重度蒙版**：70-80%（突出文字）

### 虚化强度
- **轻度虚化**：10-20%（保留细节）
- **中度虚化**：30-50%（作为背景）
- **重度虚化**：60-80%（完全模糊）

### 图片比例建议
- **封面图**：16:9 或 4:3，全屏或半屏
- **内容配图**：根据版式选择，建议不超过页面50%
- **背景图**：全屏，添加蒙版

## ✅ 快速检查清单

### 5秒检查
- [ ] 所有元素对齐了吗？
- [ ] 字体统一了吗？
- [ ] 配色协调了吗？

### 1分钟检查
- [ ] 标题和正文对比够明显吗？
- [ ] 相关信息靠得够近吗？
- [ ] 每页信息量控制了吗（3-5个要点）？
- [ ] 留白是否充足？
- [ ] 图片是否高清？

### 5分钟检查
- [ ] 浏览所有页面，是否"看起来像一套"？
- [ ] 字体样式（字号、行距）是否全局统一？
- [ ] 配色是否符合60-30-10法则？
- [ ] 过渡页版式是否一致？
- [ ] 图片风格是否统一？
- [ ] 逻辑流程是否清晰？

## 🚀 快速修复方案

### 问题：页面太拥挤
**3步修复**：
1. 删除不必要的信息
2. 增加页边距和元素间距
3. 拆分为多页

### 问题：配色不协调
**3步修复**：
1. 只保留1种主色
2. 降低所有颜色的饱和度
3. 使用在线工具生成配色方案

### 问题：文字层级不清晰
**3步修复**：
1. 标题字号改为正文的2倍
2. 标题使用粗体
3. 调整行间距为1.3倍

### 问题：看起来不像一套
**3步修复**：
1. 统一所有标题的字体和字号
2. 确保主色在每页都出现
3. 统一过渡页的版式

## 📱 场景应用速查

### 商务汇报 PPT
- **配色**：深蓝 + 灰色
- **字体**：微软雅黑 / Helvetica
- **风格**：简洁、专业、数据驱动
- **重点**：逻辑清晰、数据可视化

### 创意提案 PPT
- **配色**：对比色、鲜艳
- **字体**：设计感字体
- **风格**：视觉冲击、创新
- **重点**：视觉表现、情感共鸣

### 学术报告 PPT
- **配色**：深色 + 中性色
- **字体**：宋体 / 思源宋体
- **风格**：严谨、专业
- **重点**：数据、图表、引用

### 培训教学 PPT
- **配色**：温暖明亮色
- **字体**：易读字体
- **风格**：友好、清晰
- **重点**：要点突出、互动性

## 💡 记住这些

1. **对齐是专业的第一标准**
2. **少即是多，删减信息比添加更难**
3. **统一性比创意更重要**
4. **留白是设计的一部分**
5. **好的PPT不需要过多动画**
6. **配色协调比配色丰富更重要**
7. **高清图片是基础要求**
8. **每页传达1个核心信息**
9. **视觉服务于内容，不是装饰**
10. **检查、检查、再检查**
