## 📋 常见使用场景

### 场景 1：从零创建新 PPT

**你说：**
```
我需要制作一个公司年度总结PPT，大约20页，
要求专业、数据驱动、适合向高管汇报。
主题是"2024年度业绩回顾与2025年规划"。
```

**Claude 会：**
1. 询问你更多细节（配色偏好、公司品牌色等）
2. 提供专业的设计方案（配色、字体、版式）
3. 创建详细的大纲
4. 制作符合设计最佳实践的 PPT
5. 可选：使用 seedream-imagegen 生成配图

### 场景 2：优化现有 PPT

**你说：**
```
我做了一个PPT但看起来不够专业，
请帮我检查设计问题并给出改进建议。
[上传你的PPT文件]
```

**Claude 会：**
1. 分析你的 PPT 设计
2. 指出具体的设计问题（对齐、配色、排版等）
3. 提供详细的改进建议
4. 可选：直接帮你修改

### 场景 3：学习设计技巧

**你说：**
```
我想学习如何配色，请教我PPT配色的方法
```

**Claude 会：**
1. 讲解配色的基本原则
2. 提供不同场景的配色方案
3. 给出实用的配色工具推荐
4. 提供具体的配色案例

### 场景 4：生成配图

**你说：**
```
我需要为商务PPT的封面生成一张配图，
要求：深蓝色调、科技感、16:9比例
```

**Claude 会：**
1. 结合 professional-ppt-designer 的设计建议
2. 调用 seedream-imagegen skill 生成图片
3. 提供图片的后期处理建议（蒙版、裁剪等）

## 🎨 使用技巧

### 技巧 1：分步骤进行
不要一次性要求完成所有事情，可以分步骤：
1. 先讨论设计方案
2. 确定配色和风格
3. 制作部分页面
4. 调整和优化

### 技巧 2：提供详细信息
提供越多信息，Claude 的建议就越准确：
- 目标受众是谁？
- 演示场合是什么？
- 公司或个人品牌色是什么？
- 有没有参考案例？

### 技巧 3：善用检查清单
完成 PPT 后，要求使用检查清单：
```
"请使用 professional-ppt-designer 的检查清单帮我检查这个PPT"
```

### 技巧 4：学习案例
参考 skill 中的案例学习：
```
"给我看看商务汇报PPT的完整案例"
"学术报告PPT应该怎么设计？"
```

## 💡 最佳实践

### ✅ 推荐做法
1. 先明确需求和目标
2. 讨论设计方案再开始制作
3. 遵循 skill 的设计原则
4. 使用检查清单验证质量
5. 适时使用 AI 生成配图

### ❌ 避免做法
1. 不要跳过规划直接制作
2. 不要忽视设计原则
3. 不要过度装饰
4. 不要使用低质量图片
5. 不要忽略统一性检查
