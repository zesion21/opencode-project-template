# 专业 PPT 设计与制作 Skill

基于设计最佳实践，创建精美、专业、具有视觉冲击力的演示文稿。

## 核心设计原则

### 排版四大原则 (CRAP)

#### 1. 对比 (Contrast)

- **目的**: 通过视觉差异突出重点，引导观众注意力
- **实现方式**:
  - 字体大小对比：标题 vs 正文 (建议比例 2:1 或更大)
  - 字重对比：粗体标题 vs 常规正文
  - 色彩对比：深色标题 vs 浅色正文，或使用品牌主色
  - 形状对比：方形色块 vs 圆角元素
- **应用场景**:
  - 突出价格、数据、关键结论
  - 区分不同信息层级
  - 创建视觉焦点

#### 2. 重复 (Repetition)

- **目的**: 建立视觉统一性，让PPT"看起来像一套"
- **必须统一的元素**:
  - 配色系统：主色唯一，副色不超过3种
  - 字体样式：标题、正文、强调文字的字体、字号、字重保持一致
  - 色块与线条风格：圆角半径、边框粗细、阴影效果
  - 图标与图片风格：扁平/立体、配色、风格统一
  - 版式结构：导航栏位置、版心大小、页码位置
- **检查要点**:
  - 所有正文页是否使用相同基本布局？
  - 所有过渡页版式是否一致？
  - 标题字号、行距在不同页面是否相同？

#### 3. 对齐 (Alignment)

- **目的**: 建立秩序感，引导视觉流动
- **黄金法则**: 页面中任何元素的位置都不是随意的
- **对齐方式**:
  - 横排文字：居中对齐、左对齐、右对齐
  - 竖排文字：居中对齐、顶对齐、底对齐
  - 多段文字左对齐时使用"两端对齐"命令
- **高级技巧**:
  - 使用网格系统辅助对齐
  - 相关元素组对齐（如图片边缘与文字边缘对齐）
  - 利用参考线确保精确对齐
- **检验标准**: 专业PPT的第一标准就是对齐

#### 4. 亲密性 (Proximity)

- **目的**: 通过距离关系表达信息的逻辑联系
- **实现规则**:
  - 相关信息靠近，形成视觉单元
  - 不同信息组之间拉开距离
  - 文字组内行间距 < 文字组间距 < 字间距
- **具体应用**:
  - 标题与对应内容的距离小于不同内容块之间的距离
  - 使用"留白"分隔不同信息单元
  - 边距控制：页边距要足够（建议至少占页面宽度的5-8%）

### 文字层级关系

#### 层级划分原则

- **层级数量**: PPT中文字层级应控制在 1-3 个之间
- **划分方法**: 先父子级（标题-内容），再兄弟级（同级内容）
- **视觉呈现**:
  ```
  一级：主标题（最大、最醒目）
  二级：副标题或章节标题（中等大小）
  三级：正文内容（基础字号）
  ```

#### 行间距与字间距规范

- **行间距**: 1.2-1.4 倍（在段落设置中选择"多倍行距"）
- **字间距**:
  - 24号以上大标题：适当缩小字间距（-5% 到 -10%）
  - 正文：使用默认字间距
  - 避免：过度加宽字间距（破坏文字的整体性）
- **文字组间距**:
  - 文字组内行间距：1.2-1.4倍行高
  - 文字组之间距离：至少是行间距的2-3倍
  - 字间距 < 行间距 < 组间距（递进关系）

#### 字体选择

- **中文字体推荐**:
  - 商务场合：微软雅黑、思源黑体、苹方
  - 创意场合：阿里巴巴普惠体、站酷系列字体
  - 学术场合：宋体、思源宋体
- **英文/数字字体**:
  - 现代简洁：Helvetica, Arial, Roboto
  - 优雅高端：Montserrat, Gotham, Avenir
  - 数据展示：DIN, SF Pro Display
- **字体搭配原则**:
  - 不超过2-3种字体家族
  - 标题与正文可使用不同字体增强对比
  - 中英文使用不同字体时确保视觉协调

### 配色方案

#### 配色基础理论

- **主色**: 1种（品牌色或主题色）
- **副色**: 1-2种（辅助和补充）
- **强调色**: 1种（用于重点突出）
- **中性色**: 黑/白/灰（用于背景和文字）

#### 配色方案类型

1. **单色系配色**
   - 使用一种颜色的不同明度和饱和度
   - 适合：简洁、专业的商务演示
   - 示例：深蓝+浅蓝+极浅蓝+白色

2. **互补色配色**
   - 使用色相环上相对的颜色
   - 适合：需要强烈对比和视觉冲击的场景
   - 示例：蓝色+橙色，紫色+黄色

3. **类似色配色**
   - 使用色相环上相邻的颜色
   - 适合：和谐、舒适的视觉效果
   - 示例：蓝色+蓝绿色+绿色

4. **三角配色**
   - 色相环上等距离的三种颜色
   - 适合：活泼、丰富但平衡的场景
   - 示例：红色+黄色+蓝色

#### 不同场景的配色建议

- **商务汇报**: 深蓝+灰色系（稳重、专业）
- **创意提案**: 鲜艳的对比色（活力、创新）
- **学术报告**: 深色+中性色（严谨、可靠）
- **市场营销**: 品牌色+互补色（吸引眼球）
- **教育培训**: 温暖明亮的色彩（友好、易接受）

#### 配色实用技巧

- **60-30-10法则**:
  - 主色占60%（背景、大面积色块）
  - 副色占30%（次要元素、分隔）
  - 强调色占10%（重点信息、按钮）
- **色彩饱和度控制**:
  - 背景：低饱和度（10-30%）
  - 正文色块：中等饱和度（40-70%）
  - 强调元素：高饱和度（80-100%）
- **借鉴法**:
  - 使用取色器从优秀设计中提取配色
  - 参考网站：Coolors.co, Adobe Color, Color Hunt
  - 分析品牌官网、优质PPT模板的配色

### 图片使用技巧

#### 图片选择原则

- **高清原则**: 分辨率至少 1920x1080，避免模糊、像素化
- **相关性**: 图片必须与内容主题直接相关
- **风格统一**: 整套PPT的图片风格保持一致
  - 摄影风格：全部使用照片
  - 插画风格：全部使用插画
  - 扁平风格：使用扁平设计图标和图形
- **版权合规**: 使用免费图库或已授权的图片

#### 图片处理技巧

##### 1. 图片裁剪

- **黄金分割构图**: 将重点元素放在画面的黄金分割点
- **三分法构图**: 将主体放在三分线交点上
- **居中构图**: 适合对称、正式的场景
- **工具**: PPT自带裁剪功能，保持纵横比

##### 2. 图片蒙版

- **半透明蒙版**: 在图片上叠加半透明色块
  - 目的：降低图片干扰，突出文字
  - 透明度建议：30-60%
  - 颜色：使用主色或黑色/白色
- **渐变蒙版**: 使用渐变色块（如底部深色渐变）
  - 适合：图片下方放置文字的场景

##### 3. 图片虚化

- **背景虚化**: 模糊背景突出主体
  - 方法：使用"艺术效果">"虚化"
  - 强度：根据需要调整（轻度10-20%，重度50-70%）
- **高斯模糊**: 将图片作为背景时使用高强度模糊

##### 4. 图片去色

- **黑白效果**: 将图片转为黑白
  - 适合：营造专业、严肃的氛围
  - 可与局部彩色元素对比
- **双色调**: 使用特定颜色滤镜
  - 适合：统一图片色调与主题色

#### 图片排版布局

##### 单图布局

- **全屏背景**: 图片填满整个页面，文字叠加在上方
- **分屏布局**: 图片占半页，文字占另半页
- **小图配大字**: 图片作为装饰，文字为主体

##### 多图布局

- **网格布局**: 2x2、3x3等规整排列
  - 图片间距保持一致
  - 使用"对齐"工具确保整齐
- **拼贴布局**: 不同大小的图片组合
  - 保持视觉平衡
  - 重要图片给予更大面积
- **时间轴布局**: 横向或纵向排列展示流程
- **对角线布局**: 沿对角线排列图片

##### 图文结合

- **文字环绕**: 文字环绕图片（适合正文较多的场景）
- **图片框架**: 使用图片作为内容框架
- **图片切割**: 将文字放在图片的留白区域

#### 与 Seedream 4.0 生图配合

当需要为 PPT 生成定制化图片时，可以调用 `seedream-imagegen` skill：

##### 生图建议

1. **明确图片用途**
   - 封面图：大气、主题鲜明，16:9或4:3比例
   - 内容配图：简洁、辅助说明，根据版式选择比例
   - 背景图：低饱和度、层次丰富，适合添加蒙版

2. **提示词优化**
   - 描述风格：扁平插画、写实摄影、商务简约等
   - 指定配色：与PPT主色调保持一致
   - 明确元素：避免过于复杂的细节
   - 指定比例：根据页面布局选择16:9、4:3或1:1

3. **生成建议**
   - 批量生成2-4张，选择最佳效果
   - 使用2K分辨率（4K用于打印或高清场景）
   - 提供参考图指导风格（如已有品牌素材）

4. **后期处理**
   - 在PPT中适当裁剪以匹配版式
   - 添加蒙版提升文字可读性
   - 调整透明度、亮度、对比度以融入整体设计

##### 生图调用示例

```
请使用 seedream-imagegen skill 为我生成一张商务风格的PPT封面图，
要求：
- 风格：简约商务、科技感
- 颜色：深蓝色调为主，与我的品牌色#1a3c6d协调
- 比例：16:9
- 分辨率：2K
- 元素：抽象的科技连接线、光效，避免具体人物
```

### 视觉层次设计

#### 信息层级

- **主要信息**: 占据最大面积，最醒目（如结论、关键数据）
- **次要信息**: 中等面积，支持主要信息（如解释、背景）
- **辅助信息**: 小面积，补充说明（如注释、来源）

#### 视觉引导路径

- **Z型路径**: 标题左上→内容右上→左下→右下
- **F型路径**: 标题顶部→内容左侧纵向扫描
- **中心放射**: 中心为重点，向四周扩散
- **引导线**: 使用线条、箭头引导视线流动

#### 留白的艺术

- **页边距**: 至少占页面宽度/高度的5-8%
- **元素间距**: 不同模块之间留出足够呼吸空间
- **文字周围留白**: 避免文字紧贴边界或图片
- **"少即是多"原则**: 每页信息量控制在3-5个要点

### 常见版式布局

#### 1. 封面页

- **标准式**: 居中大标题+副标题+日期/讲者
- **分屏式**: 图片占一半，标题占另一半
- **全图式**: 全屏背景图+大标题+蒙版

#### 2. 目录页

- **列表式**: 纵向列出章节，使用编号或图标
- **卡片式**: 每个章节用独立卡片呈现
- **图文式**: 每章节配小图标或缩略图

#### 3. 内容页

- **纯文字页**: 标题+要点列表（使用项目符号或编号）
- **图文页**: 左图右文或上图下文
- **三栏布局**: 并列展示三个要点
- **对比页**: 左右对比两个方案/数据

#### 4. 数据页

- **图表为主**: 图表占70%以上，配简短说明
- **仪表盘式**: 多个小图表组合，展示关键指标
- **数字强调**: 超大数字+小字说明

#### 5. 过渡页

- **章节页**: 只显示章节标题，配简单装饰
- **引言页**: 引用名言或关键问题
- **图片页**: 全屏图片+章节标题

#### 6. 结束页

- **感谢页**: "谢谢"/"Thank You" + 联系方式
- **Q&A页**: "问答环节"提示
- **行动召唤**: "联系我们"+"扫码关注"等

## 完整制作工作流程

### 第一阶段：规划与准备

#### 1. 明确目标和受众

- 演示目的是什么？（汇报、推销、教学、提案）
- 受众是谁？（领导、客户、同事、学生）
- 期望达成什么效果？（说服、告知、激励）

#### 2. 内容梳理

- 确定主题和核心信息
- 列出主要观点（3-5个）
- 为每个观点准备支撑材料（数据、案例、图表）
- 设计逻辑结构（总-分-总、问题-方案-结果等）

#### 3. 设计规划

- 确定整体风格（商务、创意、学术、科技等）
- 选择配色方案（主色+副色+强调色）
- 确定字体（中文+英文/数字）
- 规划版式类型（需要哪些图表、图片）

### 第二阶段：创建与设计

#### 4. 阅读 pptx skill

在开始实际制作前，必须先阅读 `/mnt/skills/public/pptx/SKILL.md` 文件，了解：

- PPT 技术实现方法
- html2pptx 工作流程
- 图表和表格添加方法
- 注意事项和最佳实践

#### 5. 制作步骤

遵循 pptx skill 中的标准流程：

**a. 撰写详细大纲**

- 为每一页写出标题和内容要点
- 写出演讲者备注（每页1-3句）
- 描述每页的布局和视觉元素

**b. 设计CSS样式**

- 创建共享的 CSS 文件
- 定义 CSS 变量（颜色、字体、间距）
- 应用本 skill 的设计原则（CRAP、层级、配色）

**c. 创建HTML幻灯片**

- 为每一页创建 HTML 文件
- 应用排版四大原则
- 确保文字层级清晰
- 预留图表/表格位置（使用 placeholder）
- 嵌入共享 CSS

**d. 转换为 PowerPoint**

- 使用 html2pptx 转换 HTML 为 PPT
- 添加图表和数据
- 保存最终文件

#### 6. 图片处理

- 如需生成定制图片，调用 seedream-imagegen skill
- 对图片进行必要的处理（裁剪、蒙版、虚化）
- 确保图片分辨率足够高
- 统一图片风格

### 第三阶段：优化与检查

#### 7. 设计检查清单

**排版检查**

- [ ] 所有元素是否精确对齐？
- [ ] 标题和正文的对比是否明显？
- [ ] 文字层级是否清晰（1-3级）？
- [ ] 行间距和字间距是否合适（1.2-1.4倍）？
- [ ] 相关信息是否靠近（亲密性）？
- [ ] 留白是否充足（页边距、元素间距）？

**配色检查**

- [ ] 是否遵循60-30-10法则？
- [ ] 主色是否统一使用？
- [ ] 副色是否不超过3种？
- [ ] 配色是否符合场景（商务/创意/学术）？
- [ ] 背景和文字对比度是否足够？
- [ ] 强调色是否只用于重点信息？

**统一性检查（重复原则）**

- [ ] 字体样式是否全局统一？
- [ ] 标题字号、字重是否一致？
- [ ] 正文行间距是否统一？
- [ ] 色块、线条风格是否一致？
- [ ] 图标、图片风格是否统一？
- [ ] 版式结构是否统一（导航、版心）？
- [ ] 过渡页版式是否一致？

**图片检查**

- [ ] 图片是否高清（无模糊、像素化）？
- [ ] 图片风格是否统一？
- [ ] 图片是否与内容相关？
- [ ] 是否添加了必要的蒙版？
- [ ] 图片排版是否整齐？
- [ ] 图片与文字的间距是否合适？

**内容检查**

- [ ] 每页要点是否控制在3-5个？
- [ ] 文字是否简洁（能删则删）？
- [ ] 是否有拼写或语法错误？
- [ ] 数据来源是否标注？
- [ ] 逻辑流程是否清晰？
- [ ] 演讲者备注是否完整？

**技术检查**

- [ ] 是否有超链接需要添加？
- [ ] 动画效果是否适当（不过度）？
- [ ] 是否添加了页码？
- [ ] 文件大小是否合理（建议<50MB）？
- [ ] 字体是否已嵌入？
- [ ] 是否在目标设备上测试过？

#### 8. 测试与优化

- 在实际演示设备上预览
- 检查投影效果（颜色、对比度）
- 模拟演讲，检查内容流畅度
- 根据反馈进行调整

## 常见问题与解决方案

### 问题1：页面显得拥挤

**解决方案**：

- 减少每页信息量（拆分为多页）
- 增加页边距和元素间距
- 使用更大的字号
- 删除不必要的装饰元素

### 问题2：配色看起来不协调

**解决方案**：

- 使用在线配色工具（Coolors.co, Adobe Color）
- 参考优秀案例的配色
- 降低饱和度（避免过于鲜艳）
- 限制颜色种类（最多4-5种）

### 问题3：文字层级不清晰

**解决方案**：

- 加大标题和正文的字号差距（至少2:1）
- 使用不同字重（粗体标题、常规正文）
- 适当使用颜色区分层级
- 调整文字组间距（1.2-1.4倍行距）

### 问题4：图片与整体风格不搭

**解决方案**：

- 统一使用摄影图或插画图
- 对所有图片应用相同滤镜
- 使用统一的图片蒙版
- 考虑使用 seedream-imagegen 生成风格统一的图片

### 问题5：PPT看起来不像一套

**解决方案**：

- 检查并统一所有页面的字体样式
- 确保主色在所有页面都有出现
- 统一版式结构（导航栏、版心大小）
- 对过渡页使用完全相同的设计

### 问题6：数据图表难以理解

**解决方案**：

- 简化图表（删除不必要的网格线、图例）
- 使用颜色突出重点数据
- 添加数据标签（直接显示数值）
- 配合简短的文字说明
- 一页只展示一个核心数据洞察

## 进阶技巧

### 1. 动画效果使用原则

- **Less is More**: 动画应该服务于内容，而非装饰
- **适用场景**:
  - 揭示要点列表（逐条出现）
  - 引导视线（箭头、高亮）
  - 展示流程（步骤依次出现）
  - 强调重点（放大、颜色变化）
- **避免**:
  - 过度的飞入、旋转等花哨动画
  - 每页都加动画（会显得繁琐）
  - 动画时间过长（每个动画0.3-0.5秒为宜）

### 2. 数据可视化最佳实践

- **选择正确的图表类型**:
  - 趋势：折线图
  - 比较：柱状图
  - 占比：饼图、环形图
  - 关系：散点图
  - 流程：桑基图、流程图
- **简化图表**:
  - 删除冗余的网格线
  - 精简图例（直接标注）
  - 使用数据标签
  - 突出关键数据点
- **配色建议**:
  - 使用主题色
  - 重要数据使用强调色
  - 其他数据使用中性灰色

### 3. 讲故事的PPT结构

- **开场**: 抛出问题或痛点
- **背景**: 提供必要的背景信息
- **冲突**: 展示挑战或困难
- **方案**: 提出解决方案
- **证据**: 用数据、案例支持方案
- **行动**: 明确的行动召唤
- **结尾**: 总结和展望

### 4. 不同场景的设计侧重

**商务汇报**

- 侧重：数据、逻辑、专业性
- 配色：深色系、沉稳
- 图片：减少装饰性图片，多用图表
- 字体：无衬线字体、简洁

**创意提案**

- 侧重：视觉冲击、创新感
- 配色：大胆、对比强烈
- 图片：高质量创意图片、插画
- 字体：可使用设计感强的字体

**学术报告**

- 侧重：严谨、数据、逻辑
- 配色：中性色为主
- 图片：数据图表、实验结果
- 字体：传统字体、易读

**销售演示**

- 侧重：说服力、客户利益
- 配色：与品牌一致
- 图片：产品图、成功案例
- 字体：清晰、专业

## 总结

好的PPT设计遵循以下核心原则：

1. **CRAP四大原则**：对比、重复、对齐、亲密性
2. **简洁至上**：少即是多，每页3-5个要点
3. **层级清晰**：文字层级1-3个，视觉引导明确
4. **配色和谐**：主色+副色+强调色，遵循60-30-10
5. **图片高质**：高清、相关、风格统一
6. **统一一致**：整套PPT看起来像一个整体

制作流程：

1. 规划目标和内容
2. 设计风格和配色
3. 遵循 pptx skill 技术流程
4. 使用本 skill 的设计原则
5. 调用 seedream-imagegen 生成图片
6. 检查和优化

记住：**PPT的本质是信息传达，设计是为内容服务的。**
